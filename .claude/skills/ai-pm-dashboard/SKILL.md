# 项目仪表盘

> ⚠️ **已合并**: 本技能已合并到 `ai-pm-data`，请使用 `/ai-pm data dashboard`

## 定位

AI_PM 的**可视化中心**，提供项目全生命周期的可视化追踪。

## 迁移提示

```
旧命令: /ai-pm dashboard
新命令: /ai-pm data dashboard

过渡期: 2026-03-01 ~ 2026-03-15 (2周)
```

- **全景视图**：一眼看清项目全貌
- **进度追踪**：实时了解各阶段进展
- **产出预览**：快速查看项目产出物
- **团队协同**：支持多人协作状态展示

## 命令体系

| 命令 | 作用 | 示例 |
|------|------|------|
| `/ai-pm dashboard` | 打开主仪表盘 | - |
| `/ai-pm dashboard view {项目名}` | 查看指定项目详情 | `dashboard view meeting-assistant` |
| `/ai-pm dashboard compare {项目1} {项目2}` | 对比两个项目 | `dashboard compare p1 p2` |
| `/ai-pm dashboard export` | 导出项目报告 | `dashboard export --format=html` |
| `/ai-pm dashboard share` | 生成分享链接 | `dashboard share` |

## 项目健康度评估

### 健康度指标

| 指标 | 权重 | 计算方式 |
|------|------|---------|
| 进度健康 | 30% | 实际进度 / 预期进度 |
| 质量健康 | 30% | 基于评审问题数计算 |
| 文档健康 | 20% | 文档完整度评分 |
| 效率健康 | 20% | 单位时间产出 |

### 健康度等级

```
🟢 健康 (90-100分): 项目进展顺利，按计划推进
🟡 关注 (70-89分):  项目有轻微偏差，需要关注
🟠 警告 (50-69分):  项目有明显问题，需要干预
🔴 危险 (0-49分):   项目严重偏离，需要重大调整
```

## 集成点

### 与主控技能集成

```
项目完成阶段 → 自动更新仪表盘
用户输入命令 → 仪表盘展示当前状态
项目异常    → 仪表盘显示警告
```

### 与知识库集成

```
知识沉淀    → 仪表盘显示知识更新
模式匹配    → 仪表盘推荐相关知识
```

## Anti-Pattern

- ❌ 不要显示过多信息导致信息过载
- ❌ 不要自动刷新打断用户操作
- ❌ 不要显示敏感信息（账号密码等）
- ✅ 优先展示用户最关心的信息
- ✅ 支持自定义仪表盘布局
