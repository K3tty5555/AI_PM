# Design Context

## Target Audience
产品经理和团队，在 macOS 桌面端使用。用户熟悉 Notion、Linear 等现代工具，对设计品质有较高期望。

## Use Cases
- 管理 AI 辅助产品需求流程（分析 → PRD → 原型 → 评审）
- 查看项目阶段进展和 AI 生成内容
- 配置 AI 后端和偏好设置

## Brand Personality / Tone
**Bauhaus + Apple HIG：理性、克制、功能即美学。**
精致而不张扬，每个交互细节都有意义。像 Linear 一样高效，像 macOS 一样流畅。

## Design System (Binding Constraints)

以下约束来自 `docs/design-system.md`，**优先级高于 impeccable 默认规则**：

### Accent Color
钴蓝 `#1D4ED8`，全局唯一强调色。hover 态 `#1E40AF`。

### Background
`--background: #F8FAFF`（钴蓝调白，非纯白）
`--card: #FFFFFF`（纯白卡片在蓝调底上自然浮起）
`--bg-sidebar: rgba(234,241,252,0.90)`（毛玻璃底色同向蓝调）

### Typography
```
UI 文字：   GeistSans → -apple-system → BlinkMacSystemFont → "Segoe UI" → "PingFang SC"
长文阅读区：Lora → Georgia → serif（Markdown / 流式输出区专用，15px，line-height 1.8）
代码/终端： GeistMono → 'JetBrains Mono' → Menlo → monospace
```
GeistSans 是首选展示字体（非系统默认），-apple-system 仅作兜底。
等宽字体仅用于代码/终端输出，不用于 UI 元素。

### Spacing & Radius
- 全局圆角：8px（`rounded-lg`），卡片/Dialog 12px（`rounded-xl`）
- 主内容区：`max-w-[720px]`，padding `p-8`

### Layout
侧边栏 220px 固定宽度 + 毛玻璃效果；主内容区右侧滚动；隐藏滚动条。

### Motion
- 标准过渡：200ms `cubic-bezier(0.4, 0, 0.2, 1)`
- 进入动画：fadeInUp 250ms decelerate easing
- 按压反馈：`scale(0.97)` 100ms

### Forbidden Patterns
- ❌ `uppercase tracking-[2px]` 用于导航、按钮、标签
- ❌ `font-mono` 用于 UI 元素
- ❌ 纯黑背景 + 霓虹 accent（终末地风格）
- ❌ Card 套 Card 嵌套层级
