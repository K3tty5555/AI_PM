快速开始

安装

AI_PM 是一套 Claude Code 技能套件，无需复杂安装。

1. 克隆项目

git clone https://github.com/K3tty5555/AI_PM.git
cd AI_PM

2. 配置 Claude Code 技能

mkdir -p ~/.claude/skills
cp -r .claude/skills/* ~/.claude/skills/

3. 启动 Claude Code

claude

你的第一个项目

一条命令就行：

/ai-pm "我想做一个记账小程序，帮助年轻人管理日常开支"

AI 先问你几个问题：

  • 目标用户是谁？
  • 核心功能有哪些？
  • 和竞品有什么区别？

回答完，后面全自动：

  1. 需求分析 - 理清用户是谁、痛点在哪
  2. 竞品研究 - 看同类产品都在做什么
  3. PRD 编写 - 写出完整的产品需求文档
  4. 原型设计 - 出可点击的网页原型

更多工作流示例

团队协作模式

人多力量大，复杂项目派团队上：

/agent-team "开发一个智能客服系统" --mode=serial

竞品分析模式

想看竞品长什么样？直接抓：

/playwright-cli open https:// competitor.com

经验复用模式

做过类似的？从知识库里翻：

/ai-pm-knowledge search "登录注册设计"

规范驱动模式

有设计规范？先导入，再出图：

/ai-pm-config ui upload enterprise ./design-spec.pdf
/ai-pm-prototype --spec=enterprise
