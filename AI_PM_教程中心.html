AI PM
GETTING STARTED

快速开始

两种使用方式，根据你的偏好选择。Claude Code 命令行原生集成，桌面客户端可视化操作。

01
克隆项目

把项目拉到本地。没用过命令行？把下面这行复制到终端回车即可。

$ git clone https://github.com/K3tty5555/AI_PM.git
02
打开目录

用 Claude Code 打开 AI_PM 目录（File → Open Folder）。还没装？前往 claude.ai/code。

> File → Open Folder → AI_PM/
03
输入需求

用 /ai-pm 命令描述你的想法。AI 会先确认方向，然后自动推进到 PRD 和原型。

/ ai-pm "帮年轻人记账的小程序"
01
下载安装

从 GitHub Releases 下载对应平台的安装包，双击安装。

→ github.com/K3tty5555/AI_PM/releases
02
配置 AI 后端

设置中选择：Anthropic API（填 Key）/ OpenAI 兼容接口 / Claude CLI（复用本机登录）。

~ 设置 → AI 后端 → 选择模式
03
创建项目

点击「新建项目」，输入需求。每个阶段支持「先聊聊」模式——和 AI 讨论达成共识后再生成。

+ 新建项目 → 输入需求 → 开始
SKILLS

技能列表

21 个项目技能，覆盖完整产品流程、扩展工具与配置管理。命令直接复制到 Claude Code 输入框运行。

/ai-pm 核心
AI PRODUCT MANAGER

不知道从哪里开始就从这里开始。告诉 AI 你想做什么，它会一步步引导你把需求、竞品、PRD、原型都跑完。中途可以随时暂停，下次接着来。

/ai-pm "帮年轻人记账的小程序"
/ai-pm-analyze 核心
REQUIREMENTS ANALYSIS

把模糊的需求问清楚。会问你目标用户是谁、痛点在哪、想做哪些功能，整理成分析报告存下来。产出：02-analysis-report.md

/ai-pm-analyze "企业内部审批系统"
/ai-pm-research 核心
COMPETITIVE RESEARCH

扫一遍市场上的同类产品，看他们做了什么、没做什么，帮你找到差异化的切入点。产出：03-competitor-report.md

/ai-pm-research "记账类 App"
/ai-pm-story 核心
USER STORY WRITER

把功能拆成用户故事格式，每条附带验收标准，可以直接给开发团队用。产出：04-user-stories.md

/ai-pm-story
/ai-pm-prd 核心
PRD GENERATOR

把前面分析的结论整合成一份完整 PRD。默认输出 Markdown，按需导出 DOCX/PDF/分享页；客户端可评分、对比版本、让 AI 插图。产出：05-prd/05-PRD-v1.0.md

/ai-pm-prd --export=pdf
/ai-pm-prototype 核心
PROTOTYPE GENERATOR

根据 PRD 生成一个能点击的 HTML 原型。可读取设计规范或既有代码的设计指纹，生成后做完整性审计。双击 index.html 直接预览。

/ai-pm-prototype
/ai-pm-review 核心
REQUIREMENTS REVIEW

模拟一场评审会。产品、设计、前端、后端、测试、运营——六个角色同时开口，从各自立场找问题、给意见。产出：08-review-report-v1.md

/ai-pm-review --round=2
/ai-pm-driver 核心
PM QUALITY GATE

评审前的产品经理守门。按 KettyWu 风格检查 PRD 的目标、边界、指标、表达和可落地性，先把硬伤挡在评审会前。

/ai-pm-driver 05-prd/05-PRD-v1.0.md

/ai-pm-priority

/ai-pm-weekly

DESKTOP CLIENT

桌面客户端

独立桌面应用，macOS / Windows 双平台，v0.4.4。

核心体验

先聊聊模式
每个阶段可先和 AI 多轮讨论，达成共识后再生成文档。像真人协作一样。
分段渐现
内容按段落 fade-in 渐次呈现，不再是等半天然后突然全出来。可随时 Esc 跳过。
工具调用进度
AI 搜索网页、执行脚本时，实时显示工具状态和 Token 费用。
暗色模式
深色界面一键切换，护眼同时保持所有功能可用。⌘D 快捷切换。
10 个阶段页
需求速评、收集、分析、研究、故事、PRD、埋点、原型、评审、复盘，按项目流转。

可视化

项目 Dashboard
卡片视图，搜索/过滤/收藏/进度条，所有项目状态一目了然。
故事拖拽排序
StoryBoard 卡片拖拽，双击编辑，像 Trello 一样管理用户故事。
原型设备模拟
生成的原型可在 iPhone / iPad / Laptop / Desktop 四种设备下预览。
动效档位选择
原型三档动效强度：低·克制（B端）/ 中·平衡（C端）/ 高·丰富（营销页）。
PRD 工作台
目录、Mermaid、版本列表、差异对比、PRD 评分、助手面板和 AI 插图集中在同一页。

导出能力

PDF 导出
15 套封面模板，Chrome 无头渲染，一键生成专业 PDF 文档。
DOCX 导出
13 套配方风格，可直接导入飞书/Lark，格式完美适配。
PPT 导出
18 套配色方案按行业匹配，5 种页面类型，大纲可调整后一键生成。
分享页
独立 HTML 分享页，发给干系人直接浏览器打开，无需安装任何工具。

功能广场

图片与视觉
AI 文生图、文章封面、配图、信息图、小红书图片、知识漫画、GIF 贴纸和图像分析。
视频与音频
AI 视频生成、语音音乐、图片幻灯片和 YouTube 字幕等多媒体工作流。
文档与内容
PPT、PDF、Word、Excel、翻译、Markdown 美化、微信 HTML、网页转 Markdown。
发布工具
发布公众号、微博、X 等社媒工具统一放在功能广场，和项目工作流分离。

CLI 增强

网页搜索
竞品研究时 AI 自动搜索网页获取最新信息，不再局限于训练数据。
分阶段工具白名单
竞品研究开放搜索，原型开放文件写入，其他阶段只读，安全受控。
三种 AI 后端
Anthropic API / OpenAI 兼容接口 / Claude CLI，按需选择。
AI 共享上下文
项目 memory、skills 和 agent 索引可被 Claude 与 Codex 共同读取，降低上下文漂移。
