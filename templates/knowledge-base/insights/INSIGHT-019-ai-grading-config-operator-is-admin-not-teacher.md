# 智批改配置页的操作者是科组长/管理员，不是普通教师

## 问题场景

在评审 AI批改助手 V3 PRD 时，模拟审查提出"文科教师首次使用需要新手引导"，并据此在 PRD 中加入了首次进入弹窗、引导弹窗等功能。

## 实际情况

智批改的前置配置（阅卷设置页）由**科组长/考试管理员**操作，不是普通课堂教师。

- 单校：考试管理员、单校管理员、对应学科科组长
- 联考：联考管理员、对应学科科组长、有阅卷设置权限的辅助实施人员

科组长是考试流程的熟练操作者，配置页对他们而言是管理工具而非消费型产品。"新手引导弹窗"这类冷启动设计不适合这类用户，且每场考试通常只配置一次，高频引导场景不存在。

## 适用场景

所有涉及考试阅卷设置、智批改配置、权限管理的功能评审，需先确认操作者角色是管理侧还是教学侧，不能混用"教师"这个笼统概念。
