# 均分预估四档降级逻辑

## 业务规则
组卷/出题时预估班级均分，数据优先级如下：

1. **本班本学期 ≥3 次历史正确率** → 最准，直接用
2. **同校同年级历史数据** → 次准，偏差约 5-10 分
3. **全校同学段均值** → 兜底，偏差约 10-15 分
4. **平台知识点难度系数估算** → 最不准，偏差可能 >15 分，仅用于新题/外部题

每档均分预估卡片需标注数据来源（教师需知道可信度）。完全无数据时隐藏均分预估模块，不展示无意义的数字。

## 数据来源
教育超级智能体 PRD v3.2 出题场景（7.3 章）

## 为什么重要
不同档位准确率差距极大。老师看到均分预估却不知道误差范围，会基于不可信数据做决策（出卷过难/过易）。第四档误差超 15 分时，均分预估反而会误导教师。透明标注数据来源是必须的。

## 适用范围
任何涉及考试/作业难度预估、AI 辅助出题的产品
