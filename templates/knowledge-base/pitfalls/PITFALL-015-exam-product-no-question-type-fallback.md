# 考试产品不需要为"题型标识缺失"添加降级逻辑

## 问题场景

在评审 AI批改助手 V3 PRD 时，评审意见提出"题型标识缺失时应降级按解答题处理，并在配置页展示提示"，并据此在 PRD 中写入了降级逻辑和提示文案。

## 实际情况

考试产品中，题型标识由**答题卡系统**在题目创建时确定。答题卡未定型，考试无法生成；考试无法生成，智批改配置页无法打开。因此"题型标识缺失"的场景在业务链路上不存在，为其添加降级逻辑属于防御过度，引入不必要复杂度。

## 适用场景

所有依赖答题卡系统传入题型标识的考试类产品需求评审，遇到"题型识别失败"或"题型降级"相关需求时，先确认题型标识的来源链路，避免为不存在的场景写防御逻辑。
