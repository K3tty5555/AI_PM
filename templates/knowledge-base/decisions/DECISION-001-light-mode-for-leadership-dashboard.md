# 领导汇报仪表盘选浅色而非深色模式

## 问题场景

生成数据洞察 HTML 仪表盘时，初版用了 Apple HIG 深色模式（`#0f1117` 背景）。用户反馈：深色不适合领导看，要改浅色。

## 为什么选 A（浅色）不选 B（深色）

| 维度 | 深色模式 | 浅色模式 |
|------|---------|---------|
| 汇报场景 | 投影仪/会议室表现差 | 投影和屏幕均清晰 |
| 打印输出 | 背景黑色耗墨/不可读 | 正常打印 |
| 领导习惯 | 偏工程师审美 | 符合 Office/PPT 预期 |
| 数据可读性 | 数字在深色上对比度高 | 同等可读，色彩更温和 |

**结论**：数据洞察仪表盘**默认用浅色模式**。深色备选用于工程师自用监控大屏、夜间值班场景。

## 浅色模式切换清单

CSS 变量替换：
```css
--bg-base: #f5f5f7;  --bg-surface: #ffffff;  --bg-raised: #f0f0f5;
--border: #d1d1d6;   --text-primary: #1d1d1f; --text-secondary: #6e6e73;
--blue: #0066cc;  --red: #ff3b30;  --green: #28a745;  --amber: #ff9500;
```

JS 同步修改：
```javascript
const C = {
  b: a => `rgba(0,102,204,${a})`,   r: a => `rgba(255,59,48,${a})`,
  g: a => `rgba(40,167,69,${a})`,   w: a => `rgba(255,149,0,${a})`,
  d: a => `rgba(110,110,115,${a})`, p: a => `rgba(157,91,210,${a})`,
};
const GC = 'rgba(209,209,214,.8)';
Chart.defaults.color = '#6e6e73';
```

其他注意：
- 甜甜圈图 `borderColor` 从 `'#1a1d27'` 改 `'#ffffff'`
- 气泡图标签文字从 `rgba(232,234,246,0.9)` 改 `rgba(29,29,31,0.8)`
- 热力表 `.ch/.cm/.cl/.cv` 色值全部重调

## 验证数据

汉川一中仪表盘切换后，领导汇报场景视觉效果明显改善。

## 适用场景

所有面向业务方、领导、客户的数据洞察 HTML 仪表盘。
