# 文案框架模板库

## 1. AIDA 框架
- **A** — Attention（引起注意）：用数据、痛点或反常识开头
- **I** — Interest（激发兴趣）：展示核心价值差异点
- **D** — Desire（唤起欲望）：场景化描述使用后的变化
- **A** — Action（促成行动）：明确的下一步 + 降低门槛

Prompt 模板：
```
你是一位 B 端产品营销专家。请用 AIDA 框架为「{产品名}」撰写产品介绍文案。
目标用户：{用户画像}
核心卖点：{1-3 个关键差异点}
要求：每段 2-3 句话，总字数控制在 300 字内。
```

## 2. PAS 框架
- **P** — Problem（痛点）：精准描述目标用户正在经历的问题
- **A** — Agitate（放大）：强化问题的后果和紧迫性
- **S** — Solution（解决方案）：展示产品如何解决

Prompt 模板：
```
你是用户研究专家。请用 PAS 框架为「{产品名}」撰写产品页首屏文案。
目标场景：{用户使用场景}
已知痛点：{经过验证的用户痛点}
产品方案：{产品核心功能}
要求：Problem 部分用第二人称让用户产生共鸣，Solution 部分突出可量化的效果。
```

## 3. FAB 框架
- **F** — Feature（功能特性）：这是什么
- **A** — Advantage（优势）：比竞品/现状好在哪
- **B** — Benefit（利益）：对用户具体意味着什么

Prompt 模板：
```
你是产品文案专家。请为「{产品名}」的以下功能列表生成 FAB 文案：
功能列表：
{功能1}
{功能2}
{功能3}
每个功能按 Feature → Advantage → Benefit 结构展开，Benefit 部分用数字或场景说明。
```

## 4. CTA 公式

格式：`[动作动词] + [用户得到什么] + [紧迫/简单]`

示例：
| 场景 | CTA |
|------|-----|
| 免费试用 | 立即体验，0 成本上手 |
| 付费转化 | 升级专业版，解锁全部功能 |
| 获取方案 | 获取定制方案，仅需 3 分钟 |
| 预约演示 | 预约 1 对 1 演示，看看适不适合你 |

## 5. 异议处理模板

| 异议类型 | 回应策略 | 放置位置建议 |
|---------|---------|------------|
| "太贵了" | 拆解成本 vs 收益，强调 ROI 或对比隐性成本 | 定价页/FAQ |
| "不确定有没有用" | 提供免费试用/成功案例/数据证明 | 首页下方/案例页 |
| "没时间学" | 强调上手简单（5 分钟上手/无需培训） | 功能介绍页 |
| "怕失败/迁移成本" | 提供退款保障/数据迁移支持/并行期方案 | 定价页/对比页 |

## 注入点说明

| 阶段 | 适用框架 | 用途 |
|------|---------|------|
| Phase 3（竞品研究） | AIDA | 分析竞品文案策略，找差异化切入点 |
| Phase 5（PRD 撰写） | FAB | 为每个核心功能生成规范的 FAB 描述 |
| Phase 8（评审/定稿） | CTA 审计 | 检查所有行动号召是否清晰有力 |
