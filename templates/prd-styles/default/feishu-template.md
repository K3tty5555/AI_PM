# 飞书云文档格式说明

## 概述

飞书云文档格式是针对飞书文档生态优化的 Markdown 变体，支持飞书特有的语法和样式。

## 格式对照

### 1. 标题

标准 Markdown：
```markdown
# 一级标题
## 二级标题
```

飞书优化（保留标准语法，飞书会自动识别）：
```markdown
# 一级标题
## 二级标题
```

### 2. 表格

标准 Markdown：
```markdown
| 列1 | 列2 |
|-----|-----|
| 内容 | 内容 |
```

飞书优化（支持合并单元格）：
```markdown
| 列1 | 列2 |
|-----|-----|
| 内容 | 内容 |
| ^ | 合并上方单元格 |
```

### 3. 提及用户

飞书特有语法：
```markdown
@用户名
<at id="用户ID">@显示名</at>
```

### 4. 表情

飞书表情：
```markdown
:smile: :rocket: :warning:
```

### 5. 代码块

支持指定语言：
```markdown
```javascript
console.log('hello');
```
```

### 6. 引用

标准引用：
```markdown
> 这是引用内容
```

### 7. 列表

支持任务列表：
```markdown
- [ ] 未完成任务
- [x] 已完成任务
- 普通列表项
```

### 8. 分割线

```markdown
---
```

### 9. 高亮

飞书支持的高亮：
```markdown
==高亮文本==
```

### 10. 链接

```markdown
[链接文字](https://example.com)
```
