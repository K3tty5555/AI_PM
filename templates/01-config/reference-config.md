# 参考资源配置文件

> 用于存放多个参考网页、账号密码、说明图片等信息
> 将此文件放入项目的 `07-references/` 文件夹中

---

## 📋 使用说明

1. 将此文件复制到 `07-references/reference-config.md`
2. 填写下方表格中的参考资源信息（URL、账号、密码等）
3. 将相关截图放入 `07-references/images/` 文件夹
4. 在 CLI 中执行 `/ai-pm` 或 `/ai-pm fetch` 开始分析

### 🔧 技术说明

- 使用 `playwright-cli` 进行浏览器自动化分析
- 支持 JavaScript 渲染的页面（单页应用）
- 自动保存登录会话，避免重复输入账密
- 获取页面快照（结构化元素列表）而非原始 HTML

## 💾 已保存的会话状态

AI 使用 `playwright-cli` 分析网页时会自动保存登录会话，下次分析可复用：

| 文件名 | 对应网站 | 状态 |
|--------|----------|------|
| `session-res1.json` | 资源1的URL | ⬜ 未创建 / ✅ 已保存 |
| `session-res2.json` | 资源2的URL | ⬜ 未创建 / ✅ 已保存 |
| `session-res3.json` | 资源3的URL | ⬜ 未创建 / ✅ 已保存 |

> 💡 **会话文件说明**:
> - 首次分析时，AI 会执行登录操作并保存会话
> - 后续执行 `/ai-pm fetch` 时会自动复用已保存的会话
> - 如密码更改或会话过期，删除对应的 `.json` 文件即可重新登录
